## ✨ Features & Highlights

- **Immersive 3D/WebGL Experience:** Interactive physics-based canvas powered by Three.js and custom shaders.
- **Cinematic Animations:** High-performance transition sequences and smooth scroll integration using GSAP and ScrollTrigger.
- **Modern Architecture:** Strongly-typed React + TypeScript codebase optimized for fast load times and responsive performance.
- **Advanced Projects Showcase:** Highlights elite systems such as:
  - **Drishti:** India's first intelligent custom LLM chatbot.
  - **RedxChess:** A high-performance 3640 ELO chess evaluation engine.
  - **VoteChain:** A decentralized, tamper-proof blockchain voting system.
  - **Phoenix:** A desktop AI assistant.
- **SEO & AI-Search Optimized:** Completely optimized for Google, Bing, and AI search engines (like Perplexity and ChatGPT Search) using structured JSON-LD data and semantic metadata.

## 🧰 Tech Stack

- **Frontend Core:** React, TypeScript, HTML5, Vanilla CSS
- **3D Engine:** Three.js, React Three Fiber, WebGL
- **Animation Suite:** GSAP (GreenSock), ScrollTrigger, Lenis Smooth Scroll
- **Development Tooling:** Vite, ESLint, TypeScript compiler
- **Deployment & Hosting:** Vercel

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/Gaurav1000m/My-Portfolio.git
cd My-Portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```

## 🧩 Codebase Structure & Customization

The codebase is structured to be highly modular and easy to modify:

- `src/config.ts`: **The Source of Truth.** Edit this file to update personal details, skills, experiences, and project descriptions.
- `src/components/`: Reusable components (e.g. `TubesBackground`, `LightBeamButton`, `Navbar`).
- `src/components/styles/`: Custom CSS files for styling control.
- `public/`: Assets including 3D glTF/GLB models, icons, and PDFs.
